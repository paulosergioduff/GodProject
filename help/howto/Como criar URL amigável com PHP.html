Como criar URL amigável com PHP

Publicado 17 de maio de 2016

Nesta vídeo aula será apresentado como criar URL amigável em PHP.

A URL amigável é usada para tornar a URL mais intuitiva para o usuário ler e assim melhorando também o seu posicionamento dos buscadores como o Google, Bing, Yahoo, etc.

Para criar URL amigável primeiro vamos criar e entender o HTACCESS, partiu para ver o vídeo abaixo.

A URL amigável facilita para os sistemas de busca e para o visitante compreender melhor qual URL está visitando.

Por exemplo esta fica difícil compreender: http://www.celke.com.br/blog.php?id=2

Já está fica mais fácil compreender:http://www.celke.com.br/blog/curso-de-php-mysqli-e-bootstrap

Primeiro vamos habilitar o rewrite_module do Apache, na vídeo aula é explicado como realizar a ativação do rewrite_module no WampServer.

Agora crie um arquivo .htaccess na pasta raiz do site ou sistema que você está criando. Na página adicione o seguinte conteúdo. 

RewriteEngine on
RewriteCond %{SCRIPT_FILENAME} !-f
RewriteCond %{SCRIPT_FILENAME} !-d
RewriteRule ^(.*)$ index.php?url=$1

Lendo o código acima: 

RewriteEngine on
Ativa o suporte à reescrita de URL. Isso significa que as URL poderão ser direcionadas para um determinado arquivo no apache.

RewriteCond %{SCRIPT_FILENAME} !-f
Não aplica a condição para arquivos que existam. Isso significa que as imagens, CSS, JavaScript e outros arquivos que já existem continuarão funcionando perfeitamente.

RewriteCond %{SCRIPT_FILENAME} !-d
Não aplica a condição para diretórios que existam

RewriteRule ^(.*)$ index.php?url=$1
Arquivo para onde deve ser direcionado as requisições. Isso significa que tudo que for digitado no browser, após a pasta que está o .htacces e que não exista, será direcionado 

para o arquivo index.php. E temos $1 no parâmetro url, esse "sifrao um" significa a primeira "variável" encontrada na URL.

Agora vamos tratando as requisições feita no PHP. Primeiro criamos o arquivo "index.php".

<html>
    <head>
        <title>Home</title>
        <meta charset="utf-8">
    </head>
    <body>
        <?php
            $url = (isset($_GET['url'])) ? $_GET['url']:'home.php';
            $url = array_filter(explode('/',$url));
            
            $file = $url[0].'.php';
            
            if(is_file($file)){
                include $file;
            }else{
                include '404.php';
            }            
        ?>
    </body>
</html>


Compreendendo o código acima: 

$url = (isset($_GET['url'])) ? $_GET['url']:'home.php';
A variável $url, vai receber o que estiver na variável $_GET['url'], valor recebido da URL apresentado no navegador e recebida pelo método GET. Se não tiver nada, ela recebe o valor: home.

$url = array_filter(explode('/',$url));
Agora vamos usar o explode para separa a URL pela barra "/".

$file = $url[0].'.php';
Vamos atribuir o nome do arquivo para a variável $file.


if(is_file($file)){
    include $file;
}

Se o arquivo que está na variável $file existir será incluído o arquivo. Senão existir cairá no else no qual inclui a página '404.php'.
